- SQL to create csv file automaticly

	UNION 
	(SELECT title,year,pageNumber,genreNumeric
	FROM book_tmp
	INTO OUTFILE '/Users/romyardianto/Sites/projet_s5/csv/book_tmp.csv'
	CHARACTER SET utf8
	FIELDS ENCLOSED BY '"' TERMINATED BY ';' ESCAPED BY '"'
	LINES TERMINATED BY '\r\n')



- SQL to check duplicate values of title

	SELECT title, COUNT(*) as count
	FROM book
	GROUP BY title
	HAVING COUNT(*) > 1



- SQL to delete duplicate values of title

	 DELETE n1 FROM book n1, book n2 WHERE n1.title = n2.title AND ((n1.IdBook < n2.IdBook) OR (n1.year < n2.year))
